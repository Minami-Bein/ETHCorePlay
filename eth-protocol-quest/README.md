# Ethereum Protocol Quest (MVP)

趣味闯关学习以太坊协议（EL/CL/EVM/交易生命周期/Engine API/EIP/客户端测试）。

## 快速启动

```bash
cd eth-protocol-quest
npm install
npm run dev
```

打开 `http://localhost:5173`

## 已实现（MVP）

- 世界地图（7大协议关卡）
- 关卡剧情 + 知识卡 + 测验
- 通关判定与 XP
- 关卡解锁机制
